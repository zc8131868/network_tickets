Instructions

Excel Format Requirements
  • Use the provided sample template
  • Column A: Ticket Number
  • Column B: Destination IP (can be multiple, separated by comma)
  • Column C: Protocol (tcp, udp, etc.)
  • Column D: Destination Port (can be multiple, separated by comma)
  • Column E: Vendor Name/Email (can be multiple, separated by comma)
  • Data starts from row 4
  • Save as .xlsx format
Processing Tips
  • Each row represents one VPN network policy
  • Results show ticket number, resource ID, and policy ID
  • Copy results for further use
  • Errors are shown if processing fails
  • Use "Try Again" to retry after errors
  • Multiple IPs/ports/vendors can be separated by comma, space, or newline
